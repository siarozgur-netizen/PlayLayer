# PlayLayer

Watch tutorials or videos while gaming — without alt-tabbing.

![Demo](assets/demo.gif)

## Why PlayLayer?

When playing games like Minecraft, tutorials often require constant alt-tabbing between the game and YouTube.

PlayLayer solves this by placing a lightweight video overlay directly on top of the game.

## Features

• Click-through overlay (PASS mode)  
• Interactive mode for controlling video  
• Global hotkeys  
• Instant video search  
• Works with borderless fullscreen games  

## Hotkeys

Ctrl + Alt + C → open guide  
Ctrl + Alt + O → toggle overlay  
Ctrl + Alt + S → search video  
Ctrl + Alt + P → pause video  
Ctrl + Alt + H → go home  
Ctrl + Alt + K → rewind 10s  
Ctrl + Alt + L → forward 10s
